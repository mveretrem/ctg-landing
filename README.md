# CTG Static Site

Marketing site for Construction Technologies Group LLC, built with [Next.js](https://nextjs.org) and exported as a static site.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Edit `app/page.tsx` to update the page.

## Build

```bash
npm run build      # static export to ./out
npx serve out      # preview the static build locally
```

## Deploy

Pushing to `main` triggers an automatic Vercel deployment.
